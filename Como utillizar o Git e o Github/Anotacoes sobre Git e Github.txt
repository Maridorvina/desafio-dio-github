Passo a passo para utilização do Git

Fazer o download do Git 

<link para download do git: https://git-scm.com/downloads>

O Git CMD é como o prompt de comando normal do Windows com o gitcomando. 
Ele permite que você use todos os recursos do Git por meio da linha de 
Útil se você já estiver familiarizado com o Windows cmd e só trabalha no Windows.
O Git Bash emula um ambiente bash no Windows. 
Ele permite que você use todos os recursos do git na linha de comando, 
além da maioria dos comandos unix padrão . 
Útil se você está acostumado com Linux e quer manter os mesmos hábitos.
Git GUI é uma interface gráfica do usuário que permite que você use o 
Git sem tocar na linha de comando . É uma alternativa entre outros clientes Git. 
Como a GUI do Git é muito mínima, 
você também pode procurar outras alternativas se as GUIs lhe interessarem.
